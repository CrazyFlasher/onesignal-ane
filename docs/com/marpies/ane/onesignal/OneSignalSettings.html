setAutoRegister()method
public function setAutoRegister(value:Boolean):OneSignalSettings

Set to true to register with notification server immediately after initialization. If set to false, OneSignal.register() must be called later to successfully register with notification servers and obtain push token to receive notifications.

public function setEnableInAppAlerts(value:Boolean):OneSignalSettings

By default this is false and notifications will not be shown when the user is in your app. If set to true notifications will be shown as native alert boxes if a notification is received when the user is in your app.
